# [Personal Library](https://www.freecodecamp.org/learn/quality-assurance/quality-assurance-projects/personal-library)

## Tech Stack

**Client:** HTML, CSS

**Server:** Node, Express

**Database:** MongoDB, Mongoose

**Testing:** Mocha, Chai

**package:** Zombie

## Run Locally

```bash
git clone https://github.com/GoogolDKhan/Personal-Library.git

cd Personal-Library

npm install

npm run start
```
